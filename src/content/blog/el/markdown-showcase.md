Αυτή η ανάρτηση είναι μια ζωντανή αναφορά markdown για το ιστολόγιο `Academic-Portfolio`. Αν θέλετε κυρίως να δείτε πώς εμφανίζονται τα μπλοκ κώδικα, μεταβείτε κατευθείαν στην ενότητα "Μπλοκ Κώδικα" παρακάτω.

## Επικεφαλίδες και Ενσωματωμένη Μορφοποίηση

Το Markdown υποστηρίζει **έντονο κείμενο**, *πλάγιο κείμενο*, ***συνδυασμένη έμφαση***, ~~διαγραμμένο κείμενο~~ και ενσωματωμένο κώδικα όπως `npm run build`.

Οι τυπικοί ενσωματωμένοι σύνδεσμοι λειτουργούν, όπως [Astro](https://astro.build/), και οι σύνδεσμοι τύπου αναφοράς λειτουργούν επίσης καλά σε εκτενέστερη συγγραφή, όπως [η τεκμηρίωση του Astro][astro-docs]. Γυμνά URL όπως https://docs.astro.build αποδίδονται επίσης ως σύνδεσμοι σε αυτή τη ρύθμιση.

## Μπλοκ Παραθέσεων

> Το Markdown λειτουργεί καλύτερα όταν η πηγή παραμένει αναγνώσιμη και η αποδιδόμενη σελίδα παραμένει ευχάριστη στην ανάγνωση.
>
> Αυτό το καθιστά ιδιαίτερα χρήσιμο για τεχνικές σημειώσεις, ακαδημαϊκή συγγραφή, αρχεία αλλαγών και οδηγούς.

## Λίστες

### Μη Ταξινομημένες και Ταξινομημένες Λίστες

- Συντάξτε το περιεχόμενο της ανάρτησης στο `src/content/blog`.
- Προσθέστε επικεφαλίδες ώστε ο πίνακας περιεχομένων να έχει χρήσιμες αγκυρώσεις.
- Ελέγξτε το αποδιδόμενο αποτέλεσμα στο πρόγραμμα περιήγησης.

1. Γράψτε το frontmatter του άρθρου.
2. Προσθέστε δείγματα μπλοκ markdown.
3. Κατασκευάστε τον ιστότοπο και επιβεβαιώστε ότι η σελίδα αποδίδεται σωστά.

### Ένθετες Λίστες

- Το Markdown μπορεί επίσης να χειριστεί ένθετη δομή.
  - Αυτό είναι χρήσιμο για περιγράμματα.
  - Βοηθά επίσης κατά την τεκμηρίωση βημάτων με υπο-σημεία.
- Ο στόχος εδώ είναι απλά να διευκολύνει τον έλεγχο της απόδοσης.

## Μπλοκ Κώδικα

Τα ακόλουθα παραδείγματα διευκολύνουν τη σύγκριση διαφορετικών στυλ μπλοκ κώδικα με περίφραξη στο τρέχον θέμα. Κάθε μπλοκ εμφανίζει τώρα μια ετικέτα γλώσσας στα αριστερά και ένα κουμπί αντιγραφής στην επάνω δεξιά γωνία.

```bash
npm install
npm run dev
npm run build
```

```ts
type Publication = {
  title: string;
  year: number;
  tags: string[];
};

const recentPublications = (items: Publication[]) =>
  items.filter((item) => item.year >= 2024);
```

```astro

---

  {highlights.map((item) => <li>{item}</li>)}
</ul>
```

## HTML Μέσα σε Markdown

Ακατέργαστο HTML είναι επίσης διαθέσιμο όταν χρειάζεστε κάτι που το markdown από μόνο του δεν εκφράζει καθαρά.

<details>
  <summary>Αναπτυσσόμενη σημείωση</summary>
  <p>Αυτό είναι χρήσιμο για αποκαλύψεις, ενότητες συχνών ερωτήσεων ή προαιρετικές σημειώσεις υλοποίησης μέσα σε μια ανάρτηση.</p>
</details>
